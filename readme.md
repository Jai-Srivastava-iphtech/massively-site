## Introduction  

Massively is a pixel-perfect **HTML & CSS UI Clone** of the popular **Massively template by HTML5 UP**, focused on practicing real-world frontend layout, typography, spacing, and component styling using **pure HTML5 and CSS3 only (no JavaScript)**.

This project demonstrates a **real-world static website structure** with multiple pages such as **Landing Page**, **Generic Page**, and **Elements Reference Page**, all connected through clean navigation and reusable UI components.

---

## Screens Included  

- **Landing Page (Home Screen)**: Hero intro with call-to-action and featured posts  
- **Generic Page**: Article-style content page with hero image  
- **Elements Reference Page**: UI component showcase (buttons, lists, tables, forms, icons, images)  
- **Navigation Bar**: Sticky header navigation  
- **Footer Section**: Contact form and social information  

---

## Features  

### Landing Page (Home Screen)  

- Acts as the main entry point of the website  
- Full-screen hero intro section  
- Smooth scroll to content using anchor navigation  
- Featured post section with large hero card  
- Blog-style post grid layout  
- Pagination UI  
- Fully responsive layout  

---

### Generic Page  

- Article-style layout with:  
  - Hero image  
  - Headings and subtitles  
  - Readable typography  
- Clean content spacing  
- Matches original Massively layout style  

---

### Elements Reference Page  

- Showcases reusable UI components including:  
  - Buttons (default, primary, large, small, fit, icon)  
  - Lists (ordered, unordered, alternate, icons)  
  - Tables (default and alternate styles)  
  - Forms (inputs, select, textarea, radio, checkbox)  
  - Images (fit, left, right, grid)  
  - Blockquotes and boxes  
- Designed as a UI style guide for the project  

---

### Navigation  

- Sticky navigation bar  
- Active tab highlighting  
- Hover effects matching the original design  
- Consistent navigation across all pages  

---

### Footer  

- Contact form with styled inputs  
- Address, phone, email layout using flex alignment  
- Social media icons with hover effects  
- Pixel-matched spacing and alignment
